About the Project

This project leverages Machine Learning to predict a student's academic performance based on various social, personal, and academic factors. By analyzing data patterns, the model helps identify which students may need additional support to succeed.

The model has been trained on the widely used UCI Student Performance Dataset. Multiple algorithms such as Logistic Regression, Random Forest, Gradient Boosting, XGBoost, and CatBoost were evaluated, with ensemble models showing the highest prediction accuracy.

What Does This App Do?

This web application takes details of a student as input (like age, study time, and past failures) and predicts whether the student is likely to perform well academically.

Such predictions can help teachers, parents, and students make informed decisions, take corrective actions, and ultimately improve academic outcomes.

Get Started

To try it out, head over to the Prediction Page and enter the required details. The system will return a performance prediction instantly.
